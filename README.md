# microDIE

## microDIE Website

Sitio web de microDIE (Departamento de Ingeniería Eléctrica)

## Descripción

Este repositorio contiene el código fuente del sitio web oficial de microDIE. El sitio incluye información sobre:
- **Inicio**: Presentación del proyecto
- **Equipo**: Miembros y colaboradores
- **Proyectos**: Iniciativas y trabajos en desarrollo
- **Publicaciones**: Investigaciones y artículos

## Características

- Sitio web estático optimizado
- Diseño responsive y moderno
- Estructura HTML, CSS y JavaScript
- Hospedado en GitHub Pages
- Dominio proporcionado por UCU

_Por cualquier consulta con respecto al dominio consultar con Infraestructura UCU_

# Estructura del Proyecto

## Requisitos

- Node.js (opcional, para desarrollo local)
- Navegador web

## 📦 Instalación y Uso

### Clonar el repositorio

```bash
git clone https://github.com/uDIE-UCU/uDIE-website.git
cd uDIE-website
```
### Visualizar la página

Abrir el archivo 'index.html' en tu navegador.

## Sitio en línea

El sitio está disponible en https://die.ucu.edu.uy/ y en https://microdie.uy/

## Cómo contribuir

Las contribuciones son bienvenidas. Para reportar problemas o sugerir mejoras:

1. Abre un [issue](https://github.com/uDIE-UCU/uDIE-website/issues)
2. Realiza un fork del repositorio
3. Crea una rama para tu cambio (git checkout -b feature/mejora)
4. Commit tus cambios (git commit -am 'Añade mejora')
5. Push a la rama (git push origin feature/mejora)
6. Abre un Pull Request
